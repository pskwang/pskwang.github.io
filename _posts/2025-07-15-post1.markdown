# ✅ 4. 환경변수 설정

## 🧩 환경변수란?

- 절대경로를 매번 입력하지 않기 위해 설정
- 파일명만 입력해도 어디서든 실행 가능하게 만드는 설정

## 🧭 설정 방법

- **위쪽**: 계정 단위 설정 (개별 사용자)
- **아래쪽**: 시스템 전체 설정 (모든 사용자)

## 📦 중요 폴더

- `bin/` 폴더:
    
    → binary (이진 파일), 컴파일이 완료된 실행 파일 모음 (.exe 포함)
    

## 🔎 참고

- 자바 설치 후 인스톨러가 `PATH` 등록 자동화하지만, **직접 등록해야 안정적**
- 블로그 검색: "자바 path 설정" → 많은 자료 존재

## 5-2. Rust 언어

- DUFS 서버는 Rust 언어 기반으로 개발됨
- Rust 설치 및 환경 구성 필요

### ⚙️ 실행 및 저장 팁

- `.exe` 파일 실행 시:
    1. `Alt + D`
    2. `cmd` 입력 후 Enter
- **다른 이름으로 저장**:
    
    `Alt + F + A`
